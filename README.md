# UploadQlikS3
This script is meant to be a scheduled uploader from Qlik Sense Enterprise apps to a S3 bucket.
Use at your own risk, 
very early development stage, contribute if you can

Regards

You **must** install
* https://github.com/ahaydon/Qlik-Cli
